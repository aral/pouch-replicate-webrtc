> Replicate a PouchDB over a WebRTC DataChannel.


## Install

```sh
$ npm install --save pouch-replicate-webrtc
```


## Usage

```js
var pouchReplicateWebrtc = require('pouch-replicate-webrtc');

pouchReplicateWebrtc('Rainbow');
```
